## 背景:我们已凿到的框架(你需要这些新上下文,之前文件没有)
**开发不是一跳,是一条级联信道(马尔可夫链)**:
```
人类意图(最高熵) ─T1→ Spec ─T2→ Code ─T3→ 物理结果(编译/测试/dogfood)
   ▲                                                    │
   └──── T3 反馈:残余熵路由回 T1(改spec) 或 T2(改code) ┘
```
- 数据处理不等式(DPI):信息沿链只减不增,T1 丢的意图下游捞不回 → **T1 是最高杠杆站**。
- **信道 = agent,不是 model**。model(如 Fable5)是封闭固定 SNR;**agent = model + harness,harness 可改**,靠 harness 抬 SNR(进信道前降输入熵 / 出信道后纠错码)。我们全部工程面 = harness。
- **ΔH 真值不可算**(条件柯氏复杂度 + 离散无梯度,你上一轮已证),所以工程上只能用**可测估计器**。主控已给三个候选(你要去找更好/更贴每一站的):

  - (B) 物理锚:测试/覆盖/门禁增量(来自 T3,最难刷)
  - (C) 判官模型 EIG:EIG=H(θ)−E_y[H(θ|y)],用强 critic 模型估残余不确定性前后差
